
Descrição:

Desenvolvi um projeto de monitoramento de perfis lineares (MPL) que implementou uma carta de controle 
Shewhart e uma carta de controle EWMA para garantir a qualidade e a estabilidade dos processos industriais. Utilizei R para 
análise de dados e visualização de resultados. Este projeto envolveu a identificação de anomalias nos perfis lineares, 
permitindo uma detecção precoce de desvios do processo e tomada de decisões proativas para manter a qualidade do produto.

Responsabilidades e Conquistas:

- Projetei e implementei algoritmos de carta de controle Shewhart e EWMA em R para o monitoramento eficaz dos perfis lineares.
- Analisei e visualizei dados de processos industriais para identificar tendências e desvios usando técnicas estatísticas avançadas.

Tecnologias Utilizadas:

- R (para análise de dados e visualização)
- Cartas de controle Shewhart e EWMA

Resultados:

- Redução das falhas do processo por meio da detecção precoce de anomalias nos perfis lineares.
- Otimização da eficiência do processo ao melhorar a estabilidade e a qualidade do produto.
